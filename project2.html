USA Housing Analysis

Data Overview

The dataset provides basic information about each listed property, including:

  • Bedrooms: Counts the bedrooms.
  • Bathrooms: Counts the bathrooms.
  • SquareFeet: Total area in square feet.
  • ZipCode: Location of the property.
  • CrimeRate: Local crime rate.
  • SchoolRating: Rating of nearby schools.
  • Price: Property's market price.

This data helps understand property values and features across different regions.

Dataset used from Kaggle can be accessed here: USA Housing Dataset

Data Pre-processing

During data preprocessing for classification, the duplicates were eliminated first to render the data distinct. Zip codes were transformed from categorical to numeric format through Label Encoding to make them applicable within machine learning algorithms. Numeric features such as Bedrooms, Bathrooms, and SquareFeet were normalized to have a uniform scale within the data in order to enhance model performance. Price was categorized into "Low", "Medium", and "High" classes using quantile-based discretization, and hence could be employed for classification, and the first Price column was deleted because the focus now shifted to classification. These activities prepared the dataset for effective machine learning modeling by maintaining data purity, relevancy, and appropriateness for the algorithms.
